# Sravani Hospitals App

This is a modern healthcare application built with React, Vite, and tailwindCSS.

## Features
- Appointment Booking with workflows (Google Sheets/Neon DB Sync)
- Role-Based Access Control (Admin, Doctor, Patient)
- PWA Support
- 3D Hero Animations

## Environment Variables
Create a .env file with:
VITE_BASE44_APP_ID=your_app_id
VITE_BASE44_APP_BASE_URL=your_base_url
GOOGLE_SHEET_ID=...
GOOGLE_CREDENTIALS=...
DATABASE_URL=...
